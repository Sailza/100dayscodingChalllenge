PROBLEM : To show default constructor with null values 

SOLUTION 
The default constructor is used to provide the default values to the object like 0, null, etc., depending on the type.
In the Program , We are not creating any constructor so compiler provides you a default constructor. 
Here 0 and null values are provided by default constructor.

in our program code we create one int type and String type 
For int it will display 0 value 
and for string type it will display null value
