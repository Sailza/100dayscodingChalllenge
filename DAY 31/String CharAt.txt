PROBLEM - TO FIND THE CHARACTER AT GIVEN POSITION IN A STRING TAKING INPUT FROM USER IN JAVA LANGUAGE 

EXPLANATION 
The charAt() method returns the character at the specified index in a string.
The index of the first character is 0, the second character is 1, and so on.

POSSIBLE TEST CASE 
TEST CASE -1 
WHEN STRING IS EMPTY 
SOLUTION = IT WILL RETURN NULL VALUE 

TEST CASE -2 
WHEN CHARACTER IS PRESENT IN GIVEN STRING
SOLUTION= IT WILL RETURN VALUE AT GIVEN STRING 

TEST CASE -3 
WHEN CHARACTER IS NOT PRESENT IN  GIVEN STRING 
IT WILL RETURN -1 VALUE 

TEST CASE-4 
WHEN WE PASS INTEGER VALUE INSTEAD OF A STRING 
EXCEPTION OCCURS
